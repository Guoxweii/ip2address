# Ip2address

    instance = Ip2address::Core.new(ip)

## Installation

Add this line to your application's Gemfile:

    gem 'ip2address'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ip2address

## Usage

use taobao ip api to get info
( http://ip.taobao.com/ )
